Αφού κάνουμε μια δοκιμαστική κλήση του Covid-19 web service μέσω
του Postman, μπορούμε να προχωρήσουμε στην ανάπτυξη ενός απλού
προγράμματος Java, το οποίο:
– Θα καλεί το συγκεκριμένο web service με τις ίδιες παραμέτρους που
είδαμε παραπάνω,
– Θα λαμβάνει το Json response
– Θα μετατρέπει το Json response στο σχετικό αντικείμενο java
(αποσειριοποίηση του json response)
– Θα εμφανίζει τα στοιχεία του αντικειμένου στην κονσόλα
• Δημιουργούμε ένα Maven project με τα ακόλουθα στοιχεία:
GroupID: RestAccessor
ArtifactID: WebServiceCall

-----

Πριν προχωρήσουμε στη συγγραφή του προγράμματος που θα κάνει
consume το web service θα πρέπει πρώτα να δημιουργήσουμε μια κλάση
java (POJO class – Plain Old Java Object Class), έστω την CovidCountry, την
οποία θα χρησιμοποιήσουμε στο πρόγραμμα για την αποσειριοποίηση του
json response που μας επιστρέφει το web service.
